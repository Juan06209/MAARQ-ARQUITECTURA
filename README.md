# MAARQ Arquitectura — Landing

Landing page sencilla para estudio de arquitectura.

Cómo ver localmente:

```bash
cd "c:\Users\Imaginamos\Desktop\proyecto aqruitecta"
python -m http.server 8000
# abrir http://localhost:8000
```

Cambios incluidos:
- SEO / Open Graph meta tags
- Lazy-loading en imágenes
- Lightbox modal para ver imágenes del portafolio
- Validación simple del formulario antes de abrir WhatsApp
- Iconos sociales, favicon, robots.txt y sitemap.xml
- Animaciones suaves y mejoras de accesibilidad

Sugerencias: optimizar imágenes (webp), agregar analítica y páginas de proyecto para SEO.
